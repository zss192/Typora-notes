10.docker部署项目

- 首先根据dockerfile制作镜像(注意后面有个 . )

  ```bash
  docker build -t appservice .
  ```

- docker images会查看到存在名为appservice的镜像

- 启动容器

  ```bash
  docker run --env PORT=3011 -p 3011:3011  appservice #--env设置环境变量 -p指定端口映射
  ```
